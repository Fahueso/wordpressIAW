    El Editor Gutenberg en WordPress (2025)¶

    El editor Gutenberg —oficialmente Editor de Bloques— es el corazón de WordPress desde la versión 5.0. En 2025 no solo permite crear entradas y páginas, sino también diseñar todo el sitio gracias al Full Site Editing (FSE).

    1. ¿Qué es Gutenberg?¶

    Gutenberg reemplazó al editor clásico por un sistema visual y modular donde cada elemento (párrafo, imagen, formulario, etc.) es un bloque que se puede mover, estilizar y reutilizar.

    ¿Cómo era antes?¶

    Característica Editor Clásico (pre-2018)
    Interfaz Un único cuadro de texto
    Maquetación Shortcodes o HTML manual
    Vista previa Requería recargar la página
    Columnas / Diseño Plugins o CSS adicionales

    2. Características Principales (2025)¶

    Función Descripción
    Bloques dinámicos Más de 90 bloques nativos (incluyendo consultas, navegación, comentarios).
    Block Patterns Diseños pre-hechos de bloques (hero, pricing, testimonios).
    Synced Patterns Bloques reutilizables que se actualizan globalmente.
    Global Styles Cambia colores, tipografías y espaciado desde el editor sin tocar CSS.
    Full Site Editing (FSE) Edita header, footer, plantillas de archivo y páginas 404 con bloques.
    List View Navega y reorganiza bloques anidados fácilmente (Ctrl + Alt + O).
    Distraction Free Modo escritura limpio (Ctrl + Shift + ).
    Device Preview Vista previa móvil, tablet y escritorio en tiempo real.
    ---

    3. Cómo Usar el Editor (entradas, páginas y FSE)¶

    3.1 Entradas / Páginas¶

    1. Entradas > Añadir nueva (o Páginas).
    2. Inserta bloques con el botón + o escribiendo /.
    3. Personaliza desde la barra lateral derecha (Color, Tipografía, Espaciado, etc.).

    3.2 Full Site Editing (FSE)¶

    1. Apariencia > Editor del Sitio.
    2. Selecciona la plantilla (p. ej. “Página de inicio”, “Single”, “404”).
    3. Añade bloques de sitio (Logo, Navegación, Consulta de entradas, etc.).
    4. Aplica Global Styles desde el icono de estilos (icono de círculo dividido).

    4. Tipos de Bloques Más Usados (2025)¶

    Categoría Ejemplos
    Texto Párrafo, Encabezado, Lista, Cita.
    Medios Imagen, Galería, Carrusel, Video, Audio.
    Diseño Columnas, Grupo, Fila, Separador, Espaciador.
    Consultas Loop de entradas, Lista de páginas, Comentarios.
    Tema Logo del sitio, Navegación, Título del sitio, Menú.
    Widgets Búsqueda, Categorías, Últimos comentarios.
    Incrustados YouTube, Twitter, Spotify, Mapas, TikTok.
    ---

    5. Ventajas del Editor en 2025¶

    1. No-code total: Maquetar sin CSS ni HTML.
    2. Rendimiento: Código limpio y carga solo lo necesario.
    3. Accesibilidad: Bloques con ARIA y navegación por teclado.
    4. Escalabilidad: Desde blogs hasta tiendas (WooCommerce).
    5. Futuro-proof: WordPress Core prioriza mejoras en Gutenberg.

    6. Consejos Prácticos (2025)¶

    • Atajos

      • / → buscador rápido de bloques.
      • Ctrl + Shift + D → duplicar bloque.
      • Ctrl + Alt + O → List View.

    • Optimización

      • Usa Synced Patterns (Reusable Blocks) para botones o banners que se repiten.
      • Activa carga diferida en imágenes (ajuste nativo del bloque Imagen).

    • Accesibilidad

      • Añade texto alternativo desde el panel lateral.
      • Usa encabezados en orden jerárquico (H1 → H2 → H3).

    • Rendimiento

      • Evita bloques pesados en la cabecera (carruseles, vídeos autoplay).
      • Revisa el tiempo de carga con el plugin Query Monitor.
